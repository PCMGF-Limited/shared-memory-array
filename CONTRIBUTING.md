# Contributing

Thanks for contributing!

## Quick Start

1. Fork the repo
2. Clone it: `git clone https://github.com/yourusername/shmanager.git`
3. Install: `pip install -e .[dev]`
4. Make changes
5. Run tests: `pytest`
6. Format code: `black src/ tests/`
7. Push and open a Pull Request

## Guidelines

- Address open issues
- Improve existing docs
- Add tests for new features
- Write clear commit messages

## Need Help?

Open an issue to discuss ideas or ask questions.

**That's it!** We appreciate any contribution, big or small.
